## AstroBookings 
A REST API and a web application for the fictional space tourism company (AstroBookings).

Its purpose is to support its employees in their daily operations. 
Manage the rocket fleet, plan and confirm or cancel launches, as well as record passenger bookings without overbooking. 
Payments and refunds are processed through a fictional payment gateway. 

It is an MVP, so it does not require authentication or storage features.
